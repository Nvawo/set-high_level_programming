# Python - Import & Modules

This project is part of the **ALX Higher Level Programming** curriculum. It introduces the concept of importing functions and modules in Python.

## Learning Objectives

By the end of this project, I should be able to:

* Import functions from another Python file.
* Use imported functions in a program.
* Understand the purpose of the `if __name__ == "__main__":` statement.
* Follow Python coding standards (PEP 8).
* Write executable Python scripts.

## Project Structure

```text
python-import_modules/
├── README.md
├── add_0.py
└── 0-add.py
```

## Task 0: Import a Simple Function from a Simple File

The program `0-add.py` imports the `add` function from `add_0.py` and prints the result of adding two integers.

### Example

```bash
$ ./0-add.py
1 + 2 = 3
```

## Requirements

* Ubuntu 20.04 LTS
* Python 3.8+
* All files are executable.
* Code follows PEP 8 style guidelines.
* The program is protected using:

```python
if __name__ == "__main__":
```

so that it does not execute when imported as a module.
